# rustman
Cross platfrom package manager

## Description
Search and install rust binaires

## Usage
*Update all local packages*

`rustman`

*Search and install interactively a specific pacakge*

`rustman $package`

*install a pacakge*

`rustman -S $package`

*remove a pacakge*

`rustman -R $package`

*show installed pacakges*

`rustman --show-installed`

`rustman repl`
